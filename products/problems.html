Array 5문제

★

1. 두 수의 합 (Two Sum)

정수 배열 nums와 목표값 target이 주어질 때, 합이 target이 되는 두 원소의 인덱스를 반환하세요.

입력: nums = [2, 7, 11, 15], target = 9
출력: [0, 1]   // nums[0] + nums[1] = 9
힌트

해시맵에 (값 → 인덱스)를 저장하며 순회하면 O(n)에 풀 수 있습니다.

풀이 
func twoSum(_ nums: [Int], _ target: Int) -> [Int] {
    var map: [Int: Int] = [:]
    for (i, n) in nums.enumerated() {
        if let j = map[target - n] { return [j, i] }
        map[n] = i
    }
    return []
}

시간: O(n), 공간: O(n)

★

2. 배열 뒤집기

정수 배열을 제자리(in-place)에서 뒤집으세요. 추가 배열을 사용하지 마세요.

입력: [1, 2, 3, 4, 5]
출력: [5, 4, 3, 2, 1]
힌트

양 끝 포인터를 중앙으로 이동하며 swap하세요.

풀이 
func reverseArray(_ nums: inout [Int]) {
    var l = 0, r = nums.count - 1
    while l < r {
        nums.swapAt(l, r)
        l += 1; r -= 1
    }
}

시간: O(n), 공간: O(1)

★★

3. 정렬된 배열 중복 제거

정렬된 배열에서 중복을 제자리에서 제거하고, 고유 원소 개수를 반환하세요.

입력: [1, 1, 2, 2, 3]
출력: 3  // 배열 앞부분: [1, 2, 3, ...]
힌트

느린 포인터가 고유 원소 위치를 추적하고, 빠른 포인터가 순회합니다.

풀이 
func removeDuplicates(_ nums: inout [Int]) -> Int {
    guard nums.count > 1 else { return nums.count }
    var slow = 0
    for fast in 1..<nums.count {
        if nums[fast] != nums[slow] {
            slow += 1
            nums[slow] = nums[fast]
        }
    }
    return slow + 1
}

시간: O(n), 공간: O(1)

★★

4. 배열 회전

배열을 오른쪽으로 k칸 회전하세요. 제자리에서 처리하세요.

입력: nums = [1,2,3,4,5,6,7], k = 3
출력: [5,6,7,1,2,3,4]
힌트

3번 뒤집기: 전체 → 앞 k개 → 나머지

풀이 
func rotate(_ nums: inout [Int], _ k: Int) {
    let k = k % nums.count
    func rev(_ l: Int, _ r: Int) {
        var l = l, r = r
        while l < r { nums.swapAt(l, r); l += 1; r -= 1 }
    }
    rev(0, nums.count - 1)
    rev(0, k - 1)
    rev(k, nums.count - 1)
}

시간: O(n), 공간: O(1)

★★★

5. 물 담기 (Trapping Rain Water)

높이 배열이 주어질 때, 빗물이 고이는 총량을 계산하세요.

입력: [0,1,0,2,1,0,1,3,2,1,2,1]
출력: 6
힌트

투 포인터로 좌/우 최대 높이를 추적하며 물 양을 누적합니다.

풀이 
func trap(_ height: [Int]) -> Int {
    var l = 0, r = height.count - 1
    var lMax = 0, rMax = 0, water = 0
    while l < r {
        if height[l] < height[r] {
            lMax = max(lMax, height[l])
            water += lMax - height[l]
            l += 1
        } else {
            rMax = max(rMax, height[r])
            water += rMax - height[r]
            r -= 1
        }
    }
    return water
}

시간: O(n), 공간: O(1)

LinkedList 5문제

★

1. 연결 리스트 뒤집기

단방향 연결 리스트를 뒤집어 반환하세요.

입력: 1 → 2 → 3 → 4 → 5
출력: 5 → 4 → 3 → 2 → 1
힌트

prev, curr, next 세 포인터를 사용합니다.

풀이 
func reverseList(_ head: ListNode?) -> ListNode? {
    var prev: ListNode? = nil
    var curr = head
    while let node = curr {
        let next = node.next
        node.next = prev
        prev = node
        curr = next
    }
    return prev
}

시간: O(n), 공간: O(1)

★

2. 중간 노드 찾기

연결 리스트의 중간 노드를 반환하세요. 짝수 길이면 두 번째 중간 노드를 반환합니다.

입력: 1 → 2 → 3 → 4 → 5
출력: 3
힌트

slow/fast 포인터: slow는 1칸, fast는 2칸씩 이동합니다.

풀이 
func middleNode(_ head: ListNode?) -> ListNode? {
    var slow = head, fast = head
    while fast != nil && fast?.next != nil {
        slow = slow?.next
        fast = fast?.next?.next
    }
    return slow
}

시간: O(n), 공간: O(1)

★★

3. 사이클 감지

연결 리스트에 사이클이 있는지 판별하세요.

입력: 3 → 2 → 0 → -4 → (2로 연결)
출력: true
힌트

Floyd's Cycle Detection: fast가 slow를 만나면 사이클이 존재합니다.

풀이 
func hasCycle(_ head: ListNode?) -> Bool {
    var slow = head, fast = head
    while fast != nil && fast?.next != nil {
        slow = slow?.next
        fast = fast?.next?.next
        if slow === fast { return true }
    }
    return false
}

시간: O(n), 공간: O(1)

★★

4. 정렬된 두 리스트 병합

오름차순 정렬된 두 연결 리스트를 하나의 정렬된 리스트로 병합하세요.

입력: 1→2→4, 1→3→4
출력: 1→1→2→3→4→4
힌트

더미 헤드를 만들고 두 리스트에서 작은 값을 이어 붙입니다.

풀이 
func mergeTwoLists(_ l1: ListNode?, _ l2: ListNode?) -> ListNode? {
    let dummy = ListNode(0)
    var tail = dummy, p1 = l1, p2 = l2
    while let n1 = p1, let n2 = p2 {
        if n1.val <= n2.val { tail.next = n1; p1 = n1.next }
        else { tail.next = n2; p2 = n2.next }
        tail = tail.next!
    }
    tail.next = p1 ?? p2
    return dummy.next
}

시간: O(n+m), 공간: O(1)

★★★

5. K개씩 뒤집기

연결 리스트를 K개 단위로 뒤집으세요. 마지막 그룹이 K개 미만이면 그대로 둡니다.

입력: 1→2→3→4→5, k=2
출력: 2→1→4→3→5
힌트

K개를 카운트한 후 부분 뒤집기를 반복합니다.

풀이 
func reverseKGroup(_ head: ListNode?, _ k: Int) -> ListNode? {
    var count = 0, node = head
    while node != nil && count < k { node = node?.next; count += 1 }
    guard count == k else { return head }
    var prev = reverseKGroup(node, k)
    var curr = head
    for _ in 0..<k {
        let next = curr?.next
        curr?.next = prev
        prev = curr
        curr = next
    }
    return prev
}

시간: O(n), 공간: O(n/k) 재귀 스택

Stack 5문제

★

1. 괄호 검증

문자열의 괄호 (){}[]가 올바르게 짝지어져 있는지 확인하세요.

입력: "({[]})"
출력: true
힌트

여는 괄호는 push, 닫는 괄호는 스택 top과 비교 후 pop합니다.

풀이 
func isValid(_ s: String) -> Bool {
    var stack: [Character] = []
    let map: [Character: Character] = [")":"(", "]":"[", "}":"{"]
    for c in s {
        if let open = map[c] {
            guard stack.popLast() == open else { return false }
        } else { stack.append(c) }
    }
    return stack.isEmpty
}

시간: O(n), 공간: O(n)

★

2. Min 스택

push, pop, top, getMin을 모두 O(1)에 수행하는 스택을 설계하세요.

MinStack s; s.push(-2); s.push(0); s.push(-3);
s.getMin() → -3; s.pop(); s.getMin() → -2
힌트

보조 스택에 현재까지의 최솟값을 함께 저장합니다.

풀이 
class MinStack {
    var stack: [(val: Int, min: Int)] = []
    func push(_ val: Int) {
        let curMin = stack.last.map { min($0.min, val) } ?? val
        stack.append((val, curMin))
    }
    func pop() { stack.removeLast() }
    func top() -> Int { stack.last!.val }
    func getMin() -> Int { stack.last!.min }
}

시간: O(1) 모든 연산, 공간: O(n)

★★

3. 후위 표기법 계산

역폴란드 표기법(후위 표기법)으로 된 수식을 계산하세요.

입력: ["2","1","+","3","*"]
출력: 9  // (2+1)*3
힌트

숫자는 push, 연산자는 두 수를 pop해서 계산 후 결과를 push합니다.

풀이 
func evalRPN(_ tokens: [String]) -> Int {
    var stack: [Int] = []
    for t in tokens {
        if let n = Int(t) { stack.append(n) }
        else {
            let b = stack.removeLast(), a = stack.removeLast()
            switch t {
            case "+": stack.append(a + b)
            case "-": stack.append(a - b)
            case "*": stack.append(a * b)
            case "/": stack.append(a / b)
            default: break
            }
        }
    }
    return stack.last!
}

시간: O(n), 공간: O(n)

★★

4. 일일 온도

온도 배열이 주어질 때, 각 날에서 더 따뜻한 날까지 며칠을 기다려야 하는지 반환하세요.

입력: [73, 74, 75, 71, 69, 72, 76, 73]
출력: [1,  1,  4,  2,  1,  1,  0,  0]
힌트

단조 감소 스택: 스택에 인덱스를 저장하고 현재 온도가 더 높으면 pop합니다.

풀이 
func dailyTemperatures(_ temps: [Int]) -> [Int] {
    var result = [Int](repeating: 0, count: temps.count)
    var stack: [Int] = []  // 인덱스 스택
    for i in 0..<temps.count {
        while let top = stack.last, temps[i] > temps[top] {
            stack.removeLast()
            result[top] = i - top
        }
        stack.append(i)
    }
    return result
}

시간: O(n), 공간: O(n)

★★★

5. 히스토그램 최대 직사각형

히스토그램 막대 높이 배열이 주어질 때, 가장 큰 직사각형의 넓이를 구하세요.

입력: [2, 1, 5, 6, 2, 3]
출력: 10  // 높이5, 너비2
힌트

단조 증가 스택을 유지하며, pop될 때 직사각형 넓이를 계산합니다.

풀이 
func largestRectangle(_ heights: [Int]) -> Int {
    var stack: [Int] = [], maxArea = 0
    let h = heights + [0]
    for i in 0..<h.count {
        while let top = stack.last, h[i] < h[top] {
            stack.removeLast()
            let w = stack.isEmpty ? i : i - stack.last! - 1
            maxArea = max(maxArea, h[top] * w)
        }
        stack.append(i)
    }
    return maxArea
}

시간: O(n), 공간: O(n)

Queue 5문제

★

1. 스택으로 큐 구현

두 개의 스택을 사용하여 FIFO 큐를 구현하세요.

push(1), push(2), peek() → 1, pop() → 1, pop() → 2
힌트

inStack에 push하고, outStack이 비면 inStack에서 전부 옮깁니다.

풀이 
class MyQueue {
    var inStack: [Int] = [], outStack: [Int] = []
    func push(_ x: Int) { inStack.append(x) }
    func pop() -> Int { move(); return outStack.removeLast() }
    func peek() -> Int { move(); return outStack.last! }
    func move() {
        if outStack.isEmpty {
            while let v = inStack.popLast() { outStack.append(v) }
        }
    }
}

시간: 분할상환 O(1), 공간: O(n)

★

2. 최근 요청 수

최근 3000ms 이내의 요청 수를 반환하는 클래스를 구현하세요.

ping(1) → 1, ping(100) → 2, ping(3001) → 3, ping(3002) → 3
힌트

큐에 타임스탬프를 넣고, 범위 밖인 것은 dequeue합니다.

풀이 
class RecentCounter {
    var queue: [Int] = []
    var head = 0
    func ping(_ t: Int) -> Int {
        queue.append(t)
        while queue[head] < t - 3000 { head += 1 }
        return queue.count - head
    }
}

시간: 분할상환 O(1), 공간: O(n)

★★

3. 원형 큐

고정 크기 원형 큐를 구현하세요. enQueue, deQueue, Front, Rear, isEmpty, isFull을 지원합니다.

MyCircularQueue(3)
enQueue(1) → true, enQueue(2) → true, enQueue(3) → true
enQueue(4) → false, Rear() → 3, isFull() → true
힌트

배열 + front/rear 인덱스를 모듈러 연산으로 관리합니다.

풀이 
class MyCircularQueue {
    var data: [Int], front = 0, rear = -1, count = 0
    init(_ k: Int) { data = [Int](repeating: 0, count: k) }
    func enQueue(_ val: Int) -> Bool {
        guard count < data.count else { return false }
        rear = (rear + 1) % data.count
        data[rear] = val; count += 1; return true
    }
    func deQueue() -> Bool {
        guard count > 0 else { return false }
        front = (front + 1) % data.count; count -= 1; return true
    }
    func Front() -> Int { count > 0 ? data[front] : -1 }
    func Rear() -> Int { count > 0 ? data[rear] : -1 }
    func isEmpty() -> Bool { count == 0 }
    func isFull() -> Bool { count == data.count }
}

시간: O(1) 모든 연산, 공간: O(k)

★★

4. 프로세스 스케줄링

우선순위 배열과 위치가 주어질 때, 해당 프로세스가 몇 번째로 실행되는지 반환하세요. 가장 높은 우선순위부터 실행합니다.

입력: priorities = [2,1,3,2], location = 2
출력: 1  // 우선순위 3이 먼저 실행
힌트

큐에서 꺼낸 뒤, 더 높은 우선순위가 있으면 다시 뒤에 넣습니다.

풀이 
func processSchedule(_ priorities: [Int], _ location: Int) -> Int {
    var queue = priorities.enumerated().map { ($0, $1) }
    var order = 0
    while !queue.isEmpty {
        let first = queue.removeFirst()
        if queue.contains(where: { $0.1 > first.1 }) {
            queue.append(first)
        } else {
            order += 1
            if first.0 == location { return order }
        }
    }
    return order
}

시간: O(n²), 공간: O(n)

★★★

5. 슬라이딩 윈도우 최댓값

크기 k인 슬라이딩 윈도우가 배열을 지나갈 때, 각 위치에서의 최댓값을 반환하세요.

입력: nums = [1,3,-1,-3,5,3,6,7], k = 3
출력: [3,3,5,5,6,7]
힌트

Deque(양방향 큐)에 인덱스를 저장하며 단조 감소를 유지합니다.

풀이 
func maxSlidingWindow(_ nums: [Int], _ k: Int) -> [Int] {
    var deque: [Int] = [], result: [Int] = []
    for i in 0..<nums.count {
        while let first = deque.first, first <= i - k { deque.removeFirst() }
        while let last = deque.last, nums[last] <= nums[i] { deque.removeLast() }
        deque.append(i)
        if i >= k - 1 { result.append(nums[deque.first!]) }
    }
    return result
}

시간: O(n), 공간: O(k)

BinaryTree 5문제

★

1. 최대 깊이

이진 트리의 최대 깊이(루트에서 가장 먼 리프까지의 거리)를 반환하세요.

    3
   / \
  9  20    →  3
     / \
    15  7
힌트

재귀: 1 + max(왼쪽 깊이, 오른쪽 깊이)

풀이 
func maxDepth(_ root: TreeNode?) -> Int {
    guard let root = root else { return 0 }
    return 1 + max(maxDepth(root.left), maxDepth(root.right))
}

시간: O(n), 공간: O(h)

★

2. 좌우 반전

이진 트리의 모든 노드에서 왼쪽과 오른쪽 자식을 바꾸세요.

  4           4
 / \   →    / \
2   7      7   2
힌트

재귀적으로 left와 right를 swap합니다.

풀이 
func invertTree(_ root: TreeNode?) -> TreeNode? {
    guard let root = root else { return nil }
    let temp = root.left
    root.left = invertTree(root.right)
    root.right = invertTree(temp)
    return root
}

시간: O(n), 공간: O(h)

★★

3. 레벨 순회

이진 트리를 레벨별로 순회하여 2차원 배열로 반환하세요.

    3
   / \
  9  20    →  [[3], [9, 20], [15, 7]]
     / \
    15  7
힌트

BFS: 큐를 사용하여 레벨별로 처리합니다.

풀이 
func levelOrder(_ root: TreeNode?) -> [[Int]] {
    guard let root = root else { return [] }
    var queue: [TreeNode] = [root], result: [[Int]] = []
    while !queue.isEmpty {
        let level = queue
        queue = []
        result.append(level.map { $0.val })
        for node in level {
            if let l = node.left { queue.append(l) }
            if let r = node.right { queue.append(r) }
        }
    }
    return result
}

시간: O(n), 공간: O(n)

        let curr = stack.removeLast()

func lowestCommonAncestor(_ root: TreeNode?, _ p: TreeNode, _ q: TreeNode) -> TreeNode? {
    var node = root
    while let n = node {
        if p.val < n.val && q.val < n.val { node = n.left }
        else if p.val > n.val && q.val > n.val { node = n.right }
        else { return n }
    }
    return nil
}

func mergeKSorted(_ lists: [[Int]]) -> [Int] {
    // (값, 리스트인덱스, 원소인덱스)
    var heap: [(Int, Int, Int)] = []
    for (i, list) in lists.enumerated() where !list.isEmpty {
        heap.append((list[0], i, 0))
    }
    heap.sort { $0.0 < $1.0 }
    var result: [Int] = []
    while !heap.isEmpty {
        let (val, li, ei) = heap.removeFirst()
        result.append(val)
        if ei + 1 < lists[li].count {
            heap.append((lists[li][ei + 1], li, ei + 1))
            heap.sort { $0.0 < $1.0 }
        }
    }
    return result
}

시간: O(N log K), 공간: O(K)

class MedianFinder {
    var lo: [Int] = []  // 최대 힙 (부호 반전)
    var hi: [Int] = []  // 최소 힙
    func addNum(_ num: Int) {
        lo.append(-num); lo.sort()
        hi.append(-lo.removeFirst()); hi.sort()
        if hi.count > lo.count {
            lo.append(-hi.removeFirst()); lo.sort()
        }
    }
    func findMedian() -> Double {
        lo.count > hi.count
            ? Double(-lo.first!)
            : Double(-lo.first! + hi.first!) / 2.0
    }
}

func numIslands(_ grid: inout [[Character]]) -> Int {
    let m = grid.count, n = grid[0].count
    var count = 0
    func dfs(_ i: Int, _ j: Int) {
        guard i >= 0 && i < m && j >= 0 && j < n
              && grid[i][j] == "1" else { return }
        grid[i][j] = "0"
        dfs(i+1,j); dfs(i-1,j); dfs(i,j+1); dfs(i,j-1)

func countComponents(_ n: Int, _ edges: [[Int]]) -> Int {
    var adj = [[Int]](repeating: [], count: n)
    for e in edges { adj[e[0]].append(e[1]); adj[e[1]].append(e[0]) }
    var visited = [Bool](repeating: false, count: n)
    var count = 0
    func dfs(_ node: Int) {
        visited[node] = true
        for next in adj[node] where !visited[next] { dfs(next) }
    }
    for i in 0..<n {
        if !visited[i] { count += 1; dfs(i) }
    }

}

3. 최단 경로 (BFS)

무가중 그래프에서 시작 노드부터 각 노드까지의 최단 거리를 구하세요.

출력: [0, 1, 1, 2, 2, 3]
힌트

BFS는 레벨 단위로 탐색하므로 자연스럽게 최단 거리를 구합니다.

풀이 
func shortestPath(_ n: Int, _ edges: [[Int]], _ start: Int) -> [Int] {
    var adj = [[Int]](repeating: [], count: n)
    for e in edges { adj[e[0]].append(e[1]); adj[e[1]].append(e[0]) }
    var dist = [Int](repeating: -1, count: n)
    dist[start] = 0
    var queue = [start]
    while !queue.isEmpty {
        let node = queue.removeFirst()
        for next in adj[node] where dist[next] == -1 {
            dist[next] = dist[node] + 1
            queue.append(next)
        }
    }
    return dist
}

시간: O(V+E), 공간: O(V+E)

★★

4. 방향 그래프 사이클 감지

방향 그래프에 사이클이 존재하는지 판별하세요.

출력: true  // 0→1→2→0 사이클
힌트

DFS에서 3가지 상태(미방문, 진행중, 완료)를 사용합니다. 진행중 노드를 다시 만나면 사이클입니다.

풀이 
func hasCycleDirected(_ n: Int, _ edges: [[Int]]) -> Bool {
    var adj = [[Int]](repeating: [], count: n)
    for e in edges { adj[e[0]].append(e[1]) }
    var state = [Int](repeating: 0, count: n) // 0:흰 1:회 2:흑
    func dfs(_ u: Int) -> Bool {
        state[u] = 1
        for v in adj[u] {
            if state[v] == 1 { return true }
            if state[v] == 0 && dfs(v) { return true }
        }
        state[u] = 2; return false
    }
    for i in 0..<n { if state[i] == 0 && dfs(i) { return true } }
    return false
}

시간: O(V+E), 공간: O(V+E)

★★★

5. 위상 정렬

n개 과목과 선수 과목 조건이 주어질 때, 수강 순서를 반환하세요. 불가능하면 빈 배열을 반환합니다.

입력: numCourses=4, prerequisites=[[1,0],[2,0],[3,1],[3,2]]
출력: [0,1,2,3] 또는 [0,2,1,3]
힌트

Kahn's Algorithm: 진입 차수가 0인 노드부터 큐에 넣고 처리합니다.

풀이 
func findOrder(_ n: Int, _ prereqs: [[Int]]) -> [Int] {
    var adj = [[Int]](repeating: [], count: n)
    var indegree = [Int](repeating: 0, count: n)
    for p in prereqs { adj[p[1]].append(p[0]); indegree[p[0]] += 1 }
    var queue = (0..<n).filter { indegree[$0] == 0 }
    var order: [Int] = []
    while !queue.isEmpty {
        let node = queue.removeFirst()
        order.append(node)
        for next in adj[node] {
            indegree[next] -= 1
            if indegree[next] == 0 { queue.append(next) }
        }
    }
    return order.count == n ? order : []
}

시간: O(V+E), 공간: O(V+E)

5. LRU 캐시

get과 put을 O(1)에 수행하는 LRU(Least Recently Used) 캐시를 구현하세요.

해시맵 + 양방향 연결 리스트: 맵으로 O(1) 조회, 리스트로 순서 관리합니다.

풀이 
class LRUCache {
    class Node {
        var key: Int, val: Int, prev: Node?, next: Node?
        init(_ k: Int, _ v: Int) { key = k; val = v }
    }
    var cap: Int, map: [Int: Node] = [:]
    let head = Node(0, 0), tail = Node(0, 0)
    init(_ capacity: Int) {
        cap = capacity; head.next = tail; tail.prev = head
    }
    func get(_ key: Int) -> Int {
        guard let node = map[key] else { return -1 }
        remove(node); insertHead(node); return node.val
    }
    func put(_ key: Int, _ value: Int) {
        if let node = map[key] {
            remove(node); node.val = value; insertHead(node)
        } else {
            let node = Node(key, value)
            map[key] = node; insertHead(node)
            if map.count > cap {
                let lru = tail.prev!
                remove(lru); map[lru.key] = nil
            }
        }
    }
    func remove(_ n: Node) {
        n.prev?.next = n.next; n.next?.prev = n.prev
    }
    func insertHead(_ n: Node) {
        n.next = head.next; n.prev = head
        head.next?.prev = n; head.next = n
    }
}

시간: O(1) get/put, 공간: O(capacity)

Trie 5문제

★

1. Trie 구현

insert, search, startsWith를 지원하는 Trie를 구현하세요.

insert("apple"), search("apple") → true
search("app") → false, startsWith("app") → true
힌트

각 노드가 children 딕셔너리와 isEnd 플래그를 가집니다.

풀이 
class Trie {
    class Node {
        var children = [Character: Node]()
        var isEnd = false
    }
    let root = Node()
    func insert(_ word: String) {
        var node = root
        for c in word {
            if node.children[c] == nil { node.children[c] = Node() }
            node = node.children[c]!
        }
        node.isEnd = true
    }
    func search(_ word: String) -> Bool {
        find(word)?.isEnd == true
    }
    func startsWith(_ prefix: String) -> Bool {
        find(prefix) != nil
    }
    private func find(_ s: String) -> Node? {
        var node = root
        for c in s {
            guard let next = node.children[c] else { return nil }
            node = next
        }
        return node
    }
}

시간: O(m) per operation, 공간: O(총 문자 수)

★

2. 접두어로 단어 검색

Trie에 저장된 단어 중 주어진 접두어로 시작하는 모든 단어를 반환하세요.

words: ["apple","app","apricot","banana"]
prefix: "app" → ["apple","app"]
힌트

접두어 노드까지 이동 후, DFS로 모든 완성된 단어를 수집합니다.

풀이 
func wordsWithPrefix(_ prefix: String) -> [String] {
    guard let node = find(prefix) else { return [] }
    var results: [String] = []
    func dfs(_ n: Node, _ path: String) {
        if n.isEnd { results.append(path) }
        for (c, child) in n.children {
            dfs(child, path + String(c))
        }
    }
    dfs(node, prefix)
    return results
}

시간: O(접두어 + 결과 총 문자수), 공간: O(결과 수)

★★

3. 자동완성 Top K

검색어를 한 글자씩 입력할 때마다 가장 빈도가 높은 K개 추천어를 반환하세요.

sentences: ["i love you","island","iroman"]
times: [5,3,2]
input: "i" → ["i love you","island","iroman"]
input: " " → ["i love you"]
힌트

Trie 각 노드에 해당 접두어를 가진 상위 K개 문장을 미리 저장합니다.

풀이 
class AutoComplete {
    var trie = Trie()
    var freq: [String: Int] = [:]
    func setup(_ sentences: [String], _ times: [Int]) {
        for (s, t) in zip(sentences, times) {
            freq[s] = t; trie.insert(s)
        }
    }
    func suggest(_ prefix: String, _ k: Int) -> [String] {
        let words = trie.wordsWithPrefix(prefix)
        return words
            .sorted { (freq[$0] ?? 0) > (freq[$1] ?? 0) }
            .prefix(k).map { $0 }
    }
}

시간: O(접두어 + 결과 정렬), 공간: O(n * m)

★★

4. 사전에서 가장 긴 단어

단어 배열에서, 다른 단어들로 한 글자씩 쌓아 만들 수 있는 가장 긴 단어를 찾으세요. 동점이면 사전순이 빠른 것입니다.

입력: ["w","wo","wor","worl","world"]
출력: "world"
힌트

Trie에 삽입 후, 모든 접두어가 완성 단어인 경로 중 가장 긴 것을 찾습니다.

풀이 
func longestWord(_ words: [String]) -> String {
    let sorted = words.sorted()
    var set: Set<String> = [""]
    var result = ""
    for w in sorted {
        let prefix = String(w.dropLast())
        if set.contains(prefix) {
            set.insert(w)
            if w.count > result.count { result = w }
        }
    }
    return result
}

시간: O(n * k log n), 공간: O(n * k)

★★★

5. 와일드카드 검색

'.'이 임의의 한 문자를 대신하는 검색을 지원하세요.

addWord("bad"), addWord("dad"), addWord("mad")
search("pad") → false, search("b..") → true
search(".ad") → true, search("...") → true
힌트

'.'을 만나면 현재 노드의 모든 자식으로 분기하여 DFS합니다.

풀이 
class WordDictionary {
    class Node {
        var children = [Character: Node]()
        var isEnd = false
    }
    let root = Node()
    func addWord(_ word: String) {
        var node = root
        for c in word {
            if node.children[c] == nil { node.children[c] = Node() }
            node = node.children[c]!
        }
        node.isEnd = true
    }
    func search(_ word: String) -> Bool {
        let chars = Array(word)
        func dfs(_ node: Node, _ i: Int) -> Bool {
            guard i < chars.count else { return node.isEnd }
            if chars[i] == "." {
                return node.children.values.contains { dfs($0, i+1) }
            }
            guard let next = node.children[chars[i]] else { return false }
            return dfs(next, i + 1)
        }
        return dfs(root, 0)
    }
}

시간: O(m) ~ O(26^m) 최악, 공간: O(총 문자 수)
